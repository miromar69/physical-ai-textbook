# Welcome to Physical AI & Humanoid Robotics

This textbook is your comprehensive guide to building intelligent robots that perceive, reason, and act in the physical world. Over 12 weeks, you will progress from foundational robot communication to cutting-edge Vision-Language-Action models.

## Who This Book Is For

This textbook serves a **mixed audience**:

- **University students** taking a structured course in robotics and AI
- **Industry professionals** transitioning into physical AI and humanoid robotics
- **Self-learners** exploring robotics as a hobby or career pivot

Each chapter is written at an intermediate level by default. If you're logged in, you can click **"Personalize"** on any chapter to adapt the content to your specific background.

## How to Use This Book

### Sequential Path
Follow the modules in order (1 → 2 → 3 → 4) for a structured 12-week learning experience. Each module builds on concepts from the previous one.

### Reference Path
Jump directly to the module or chapter that interests you. Each chapter lists its prerequisites at the top so you know what background knowledge is assumed.

## What You Will Learn

By the end of this textbook, you will be able to:

1. **Build robot systems** using ROS 2 — the industry-standard robot middleware
2. **Simulate robots** in Gazebo and Unity, creating digital twins for safe experimentation
3. **Train robot intelligence** using NVIDIA Isaac for reinforcement learning
4. **Understand VLA models** — the frontier of robot learning that combines vision, language, and action
